# Fonts

Place the .woff2 font files here. Download from Google Fonts:
- https://fonts.google.com/specimen/Syne  → Download family → extract woff2
- https://fonts.google.com/specimen/DM+Mono → Download family → extract woff2

Required files (rename to match exactly):
  Syne-Regular.woff2
  Syne-Medium.woff2
  Syne-SemiBold.woff2
  Syne-Bold.woff2
  Syne-ExtraBold.woff2
  DMMono-Light.woff2
  DMMono-LightItalic.woff2
  DMMono-Regular.woff2
  DMMono-Medium.woff2
